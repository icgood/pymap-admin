pymapadmin¶

pymapadmin.local¶

pymapadmin.local.get_admin_socket(*, mkdir=False)¶

Returns a path in gettempdir() that should be consistent between a pymap-admin client and a pymap server.

Use the $PYMAP_ADMIN_SOCK environment variable to override this behavior with an explicit value.

Parameters

mkdir (bool) – Whether the intermediate directory should be created.

Return type

str
