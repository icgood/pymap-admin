pymapadmin.local¶

pymapadmin.local.get_admin_socket(path, *, mkdir=False, user=False)¶

Returns a path that should be consistent between a pymap-admin client and a pymap server.

Use path or the $PYMAP_ADMIN_SOCK environment variable to override with an explicit path, otherwise the default is:

`bash $TMPDIR/pymap/pymap-admin.sock `

Parameters

  • path (Union[None, str, Path]) – Path provided by command-line arguments.

  • mkdir (bool) – Whether the intermediate directory should be created.

  • user (bool) – If True, the default path will be in the current user’s home directory.

Return type

Path

pymapadmin.local.get_token_file(path, *, mkdir=False, user=False)¶

Returns the path that should be used to read and write the auth token for use by the pymap-admin client.

Use path or the $PYMAP_ADMIN_TOKEN_FILE environment variable to override with an explicit path, otherwise the default is:

`bash $TMPDIR/pymap/pymap-admin.token `

Parameters

  • path (Union[None, str, Path]) – Path provided by command-line arguments.

  • mkdir (bool) – Whether the intermediate directory should be created.

  • user (bool) – If True, the default path will be in the current user’s home directory.

Return type

Path
